<h2>Запуск</h2>

```
npm i
npm start
selenium-standalone start
npm test```

<h2>Список логических блоков системы</h2>

<ul>
  <li>Helper для работы с гит</li>
  <li>История коммитов</li>
  <li>Файловая система</li>
  <li>Создание файла</li>
  <li>Навигация</li>
</ul>

<h2>Сценарии для логических блоков</h2>

<h3>Helper для работы с git</h3>
<ul>
  <li>Запуск гита и получение рез-та</li>
  <li>Получение истории коммитов от гита</li>
  <li>Получение списка файлов в коммите</li>
  <li>Получение содержимого файлов</li>
</ul>

<h3>История коммитов (indexController)</h3>
<ul>
  <li>Отображение истории коммитов со ссылками</li>
</ul>

<h3>Файловая система (filesController)</h3>
<ul>
  <li>Создание url в зависимости от типа файла</li>
  <li>Построение структуры файлов</li>
</ul>

<h3>Создание файла (contentController)</h3>
<ul>
  <li>Проверка типа файла</li>
  <li>Проверка что у файла есть контент</li>
  <li>Рендер файла</li>
</ul>

<h3>Навигация</h3>
<ul>
  <li>Создание пути для директории</li>
  <li>Создание пути для файла</li>
  <li>Создание хлебный крошек для главной</li>
  <li>Создание хлебный крошек для коммита</li>
  <li>Создание хлебный крошек для директорий</li>
</ul>
